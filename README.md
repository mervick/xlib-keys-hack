xlib-escape-key-hack
====================

Simulating Escape key pressing when press shift (only one, not both left and right)
without any combos.

Build
-----

```bash
$ make
```

Run (as daemon)
---------------

```bash
$ ./build/xlib-escape-key-hack
```
